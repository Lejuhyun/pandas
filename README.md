- python -m venv venv : 가상환경 생성
- source venv/Scripts/activate : 가상환경 활성화
- pip <module_name> : 모듈 설치
- pip install jupyter lab
- pip install numpy
- pip install pandas (pandas 에선 jupyter, numpy, pandas 설치)
- gitignore.com (python, window, macos) => 복사해서 .gitignore 파일에 붙여넣기
- pip freeze >> requirements.txt : 현재 설치된 모듈 리스트 저장
- pip install -r requirements.햣 txt : requirements.txt 기준으로 모듈 설치
